### Prerequisites

* Python 3.5+ 
* Tensorflow-gpu 1.1+
* Numpy 
* argparse 
* itertools 
* os

### Usage

1. **Load data**

```
$ python preprocessing.py --path 'path-where-tasks_1-20_v1-2-located'
```

2. **Run model**

```
$ python train.py
```
